# briefings/

Generated daily briefings live here. Filename: `YYYY-MM-DD.md`.

Older briefings are moved to `briefings/archive/` so they remain searchable without being loaded by default.

The briefing format and contents are defined by the canonical runtime contract at [.claude/skills/daily-briefing/SKILL.md](../.claude/skills/daily-briefing/SKILL.md). The current format is **executive-facing**: action-first, source-traceable, bullet-pointed with bolded human-readable titles, file paths and slugs in metadata under the title rather than in the title itself. Implementation and audit detail live at the end of the briefing. See the canonical section order and surfaced-item format in the skill; [2026-05-19.md](2026-05-19.md) and [2026-05-20.md](2026-05-20.md) are reference examples. The prior workflow file at `workflows/daily-briefing.md` is superseded and now redirects to the skill.
