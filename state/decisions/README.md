# state/decisions/

Open decisions awaiting Chief Executive Officer (CEO) judgment.

## Required frontmatter

- `context:`
- `decision:` — one-line question
- `options:` — list of considered options
- `needed_by:` — International Organization for Standardization 8601 (ISO 8601) date the decision must be made
- `source:`, `as_of:`, `confidence:`

Decisions move to `state/archive/` once made, with the chosen option recorded.
