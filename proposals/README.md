# proposals/

Drafted external actions awaiting Chief Executive Officer (CEO) approval. See [rules/approval-gates.md](../rules/approval-gates.md).

Filename: `YYYY-MM-DD-<short-slug>.md`. One file per proposed action. Required frontmatter is listed in the approval-gates rule.

Nothing here executes. The CEO either approves (and the action is taken outside the system, or by a future approved-execution workflow) or rejects (and the proposal is moved to `state/archive/`).
